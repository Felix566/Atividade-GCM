Essa é a primeira atividade da disciplina de Gerência de Mudanças,do curso de análise e desenvolvimento de sistemas.
A atividade é será criar um repositório,e esse repositório deverá conter as seguintes caracteristicas:
* Ter ao menos dois arquivos (de qualquer formato, exemplo: html, txt, md);
* Ter ao menos quatro commits com alterações nos arquivos do repositório;
* Ter ao menos duas branches com pelo menos um commit em cada;
* Ter ao menos uma tag com uma versão no formato de versionamento semântico (exemplo: v1.0.0);
